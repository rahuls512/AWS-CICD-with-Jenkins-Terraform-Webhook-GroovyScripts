# AWS-CICD-with-Jenkins-Terraform-Webhook-GroovyScripts

AWS DevOps automation using CI/CD with Jenkins, Terraform Webhook, and writing Infrastructure as Code (IAC) using Groovy scripts.

## Prerequisites:
1. AWS Cloud
2. Git
3. Vscode
4. GitHub
5. Terraform
6. Jenkins
